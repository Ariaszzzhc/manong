## 码农是什么？

码农是一个原生桌面应用，让 AI 编程助手直接在你的项目目录里工作。它读你的代码、改你的文件、跑你的命令、管你的任务 —— 全在一个窗口里完成。

不用开浏览器，不用来回粘贴。打开文件夹，直接开干。

## 核心能力

- **直接操作文件** — AI 在你的工作区里读、写、搜索、编辑文件。不是给建议，是直接改。
- **执行终端命令** — 装依赖、跑构建、看报错、修 bug，一个对话搞定。
- **深度思考** — 支持 Extended Thinking，复杂问题先推理再动手，思考过程可展开查看。
- **计划模式** — 只读分析模式，AI 浏览你的代码库并生成可执行的方案，不做任何修改。可在界面中切换，也可由 AI 在需要规划时自动进入。
- **子代理** — 派生子代理并行处理子任务，各自独立运行，有独立的工具集和步数限制。
- **上下文压缩** — 长对话自动管理上下文。裁剪旧的工具结果、总结历史记录，完整记录保存到 `.manong/transcripts/`。
- **自动会话标题** — AI 根据首条消息自动生成会话标题。
- **工具权限控制** — 控制 AI 能做什么。文件编辑支持 diff 预览，审批后再执行。可选默认模式、自动接受编辑、或完全跳过审批。
- **图片支持** — 在消息中附加图片，提供视觉上下文。
- **MCP 扩展** — 接入任意 [Model Context Protocol](https://modelcontextprotocol.io) 服务器，给 AI 加装新能力。支持全局和项目级配置。
- **LSP 支持** — 内置语言服务器协议集成，提供代码智能提示。
- **自定义 Skill** — 用 Markdown 写可复用的 Prompt 模板，放进 `.manong/skills/` 或 `~/.config/manong/skills/`，在聊天输入框用斜杠命令触发。
- **丰富渲染** — 语法高亮、LaTeX 公式、Mermaid 流程图、内联 diff。
- **任务管理** — 内置待办清单，多步骤任务逐项推进。
- **快捷键** — 切换视图、收起侧栏、管理会话，键盘搞定。
- **多语言** — 中英双语界面，自动检测系统语言。
- **以工作区为中心** — 会话、配置、Skill 都绑定在项目目录上，切换项目无缝衔接。

## 跟其他工具有什么不同？

| | 码农 | 浏览器工具 | 终端 Agent |
|---|---|---|---|
| 运行方式 | 原生桌面应用 | 浏览器标签页 | 终端 |
| 文件操作 | 直接读写工作区 | 复制粘贴 | 直接读写 |
| 界面 | 完整 GUI + 富文本渲染 | Web UI | TUI |
| 扩展性 | MCP + Skill | 各有不同 | 各有不同 |

码农把终端 Agent 的能力装进了桌面应用的壳里 —— 既强大，又好用。

## 快速开始

```bash
pnpm install && pnpm start
```

在 **设置** 里填入 API Key，打开一个文件夹，开始写代码。
